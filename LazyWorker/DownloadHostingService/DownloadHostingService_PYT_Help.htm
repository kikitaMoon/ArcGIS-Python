Parameter Explanation Data Type
ServiceURL Dialog Reference

The HTTP REST URL of the feature service.

Make sure the URL reachable in the client, For example, try it in the web explorer first.

Sample URL:

http://gis039.esri.local:6080/arcgis/rest/services/Hosted/Join_overlaps_sichuanyaogan_201805261731/FeatureServer/0
